# 🤖 AI노출체크

질문을 넣으면 **구글 AI 개요 · ChatGPT · Gemini**에 우리 브랜드가 노출되는지 자동으로 체크하는 프로그램입니다.

- 질문별로 여러 번 반복 측정 → **표시횟수 / 노출횟수 / 언급 순위**
- 노출 화면 **📷 캡처 증거** + **📄 답변 원문** (브랜드 형광펜 표시)
- **대시보드**: 엔진별 노출 비교, 우리 채널(홈페이지/블로그/티스토리/인스타...) 인용 분석
- CSV 내보내기

---

## 1. 설치 (처음 한 번만)

### ① 프로그램 실행 — 파이썬 설치 필요 없음!
- 이 폴더의 **`AI노출체크.exe` 더블클릭**
- 처음 실행 시 Windows가 "알 수 없는 게시자" 경고를 띄우면 → **[추가 정보] → [실행]**
- 브라우저에 http://127.0.0.1:5630 이 자동으로 열립니다 (안 열리면 직접 입력)
- 검은 콘솔 창을 닫으면 프로그램이 꺼집니다 — 측정하는 동안 켜두세요

> 파이썬이 이미 있는 PC라면 `AI노출체크_실행.bat`으로 실행해도 됩니다 (동일한 프로그램)

### ③ 크롬 확장 설치 (구글 AI개요·ChatGPT·Gemini 측정용 — 필수)
1. 크롬 주소창에 `chrome://extensions` 입력
2. 오른쪽 위 **개발자 모드** 켜기
3. **[압축해제된 확장 프로그램을 로드합니다]** 클릭 → 이 폴더 안의 **`크롬확장_AI개요체크`** 폴더 선택
4. 설치된 카드에서 **[세부정보]** 클릭 → **「시크릿 모드에서 허용」 켜기** ← 필수! (안 켜면 AI개요 측정 안 됨)
5. 퍼즐(🧩) 아이콘 → **AI개요 체크** 를 **툴바에 고정**해두면 편합니다

> ⚠️ **평소에 실제로 쓰는 크롬 프로필**에 설치하세요. 구글은 사용기록 없는 브라우저에 AI 개요를 잘 안 보여줍니다.

### ④ 로그인 준비 (측정 전 확인)
| 엔진 | 필요한 것 |
|------|----------|
| 구글 AI 개요 | 없음 (시크릿 탭에서 측정) |
| **ChatGPT** | 그 크롬에서 **chatgpt.com 로그인** 필수 (무료 계정 OK) |
| **Gemini** | 그 크롬에서 **구글 계정 로그인** (gemini.google.com 접속되면 OK) |

로그인이 안 돼 있으면 결과표에 "⚠ 로그인 필요"라고 표시됩니다.

## 2. 사용법

1. **설정** (화면 위쪽):
   - **브랜드** / **브랜드 별칭**: 우리 회사명과 검색될 수 있는 표기들 (예: `하오디자인, haod`)
   - **경쟁사**: 쉼표로 구분 — 답변에서 경쟁사 언급 순위도 같이 잡힙니다
   - **우리 도메인**: 홈페이지·블로그 주소 (예: `haodesign.co.kr, blog.naver.com/아이디`) — 대시보드 "우리 채널 인용" 분석에 사용
   - **반복 횟수**: 질문당 몇 번 측정할지 (AI 답변은 매번 달라져서 2~5회 반복 권장)
   - 텍스트 칸은 수정 후 **[설정 저장]**, 체크박스는 자동 저장
2. **질문 추가**: 여러 줄 붙여넣으면 줄마다 질문 하나씩 등록됩니다
3. **[▶ 전체 측정]** 클릭 → 자동으로:
   - 구글 AI개요: 시크릿 창이 잠깐씩 뜨면서 검색·캡처
   - ChatGPT/Gemini: 별도 창이 뜨면서 질문·답변 수집
   - 캡처 순간 창이 0.3초씩 앞으로 나옵니다 (정상 동작)
4. **결과 읽는 법**:
   - `표시 4/5` = 5번 검색 중 4번 AI 개요가 뜸 (구글 AI개요만 있는 지표)
   - `노출 2/4` = 뜬 것 중 2번 우리 브랜드 언급
   - `1위` = 언급된 업체들 중 평균 몇 번째
   - **📷** 캡처 모아보기 / **📄** 답변 원문(형광펜)+참고 출처
5. **대시보드**: 엔진별 노출 도넛, 우리 채널 인용(어느 채널이 AI 소스로 먹히는지), 인용 출처 TOP

## 3. 자주 묻는 문제

| 증상 | 해결 |
|------|------|
| 측정 눌러도 반응 없음 | 앱 상태줄 확인. "🧩 확장 연결 안 됨"이면 확장 설치/새로고침 후 페이지 새로고침 |
| ChatGPT "⚠ 로그인 필요" | 그 크롬에서 chatgpt.com 로그인 후 재측정 |
| Gemini "구글 로그인 필요" | 그 크롬에서 구글 로그인 |
| AI개요가 계속 0/N (안 뜸) | ① 그 질문엔 원래 안 뜰 수 있음 (정보형 질문에 잘 뜸) ② 단시간 과다 측정 시 구글이 잠시 억제 — 30분~1시간 후 재시도 |
| "구글 차단 감지" | 측정을 너무 자주 돌린 경우 — 잠시 쉬었다가 |
| 확장 코드가 업데이트됨 | `chrome://extensions` → 위쪽 [업데이트] 버튼 → 확장 아이콘 눌러 버전 확인 |

## 4. 알아두기

- **측정 원리**: 실제 사용자와 같은 환경(진짜 크롬)에서 검색/질문하고, 답변 **전문(全文)** 기준으로 브랜드 언급을 판정합니다
- **개인화 오염 방지**: AI개요=시크릿 탭, ChatGPT=임시 채팅(기록 안 남음), Gemini=임시 채팅 시도 — "우리가 우리 걸 자꾸 검색해서 잘 나오는" 왜곡을 차단
- AI 답변은 매번 달라질 수 있으므로 **반복 측정 + 비율**로 보는 것이 정확합니다
- 데이터는 전부 이 폴더 안에 저장됩니다 (`AI체크_데이터.json`, `캡처/` 폴더)
- 같은 질문을 너무 자주(하루 수십 회+) 돌리면 구글/OpenAI가 일시 차단할 수 있으니 적당히!
